Descripción

Emoji Cloner es un selfbot desarrollado en Python que permite copiar emojis personalizados de un servidor de Discord a otro usando un token de usuario.

El script descarga los emojis del servidor origen y los sube automáticamente al servidor destino, respetando sus nombres originales.

Características

Clona emojis estáticos y animados

Mantiene los nombres de los emojis

Funciona con múltiples servidores

Proceso automático

Basado en la librería oficial discord de Python

Requisitos

Python 3.10 o superior

Librería discord (pip install discord)

Token de usuario (selfbot)

Permisos para gestionar emojis en el servidor destino

Instalación
git clone https://github.com/tuusuario/emoji-cloner-python.git
cd emoji-cloner-python
pip install -r requirements.txt

Uso
python main.py


El script comenzará a copiar todos los emojis disponibles del servidor origen al servidor destino.
